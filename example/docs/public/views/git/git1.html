Git托管代码

laughing 2020-01-18 06:00:00 工具使用

    之前一直都是用SVN，用于微服务非常不方便，而Git是分布式的，现在学习使用Git。
     * 1、安装

    Git Windows安装地址：https://gitforwindows.org/
    小乌龟Git Windows安装地址：https://tortoisegit.org/download/

     * 2、工作流程
    类似于SVN
   &nbsp工作区：就是你在电脑里能看到的目录。
   &nbsp暂存区：英文叫stage, 或index。一般存放在 ".git目录下" 下的index文件（.git/index）中，所以我们把暂存区有时也叫作索引（index）。
   &nbsp版本库：工作区有一个隐藏目录.git，这个不算工作区，而是Git的版本库。

     * 3、Git使用
   &nbsp目前主流的Git代码托管平台
    Github：https://github.com/
    GitLab：https://gitee.com/
    Gitee：https://gitee.com/

   &nbsp修改.gitignore配置文件，过滤掉eclipse、idea的目录.idea .project，不用上传
    克隆代码命令行

   git clone https://github.com/laughingfuzihao/laughing.git
   
   git add *
   
   git commit -m “修改过滤文件”
   
   git push
